Comandos no Intellij DEA:

Comando ctrl + alt + i  atalho para ajuste na indentação do código
Comando ctrl + d duplicar a linha em baixo
Comando shift + F6 atalho para renomear um elemento ou todos de mesmo nome, sem a necessidade alterar um por um
Comando alt + insert atalho que possibilita a geração de um método
Comando ctrl + alt + o é utilizado para remover imports que não são mais usados. O Auto import pode ser configurado em Settings >> Editor >> General >> Auto import
